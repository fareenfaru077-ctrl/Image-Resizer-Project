Image Resizer Project
 Overview

The Image Resizer Project is a simple Python application that resizes multiple images automatically using the Pillow library.

The program reads images from an input folder, resizes them to a fixed dimension (800 × 600), and saves the resized images into an output folder.
Features
	•	Batch image resizing
	•	Automatic output folder creation
	•	Supports JPG, JPEG, and PNG formats
	•	Beginner-friendly and easy-to-understand code
